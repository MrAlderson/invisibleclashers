Herobrine Commands and Permissions

Command Permission Description
/herobrine help hb-ai.help Shows all commands
/herobrine attack (player name) hb-ai.attack Sets Herobrine´s attack target
/herobrine haunt (player name) hb-ai.haunt Sets Herobrine´s haunt target
/herobrine cancel hb-ai.cancel Cancel all actions against Herobrine´s currently set target
/herobrine reload hb-ai.reload Reloads config
/herobrine position hb-ai.position Gives the coordinates of Herobrine's current position
/herobrine pyramid (player name) hb-ai.pyramid Builds a pyramid near the player
/herobrine graveyard (player name) hb-ai.graveyard Teleports player to the Herobrine's Graveyard
/herobrine bury (player name) hb-ai.bury Bury player
/herobrine temple (player name) hb-ai.temple Builds a temple near the player
/herobrine heads (player name) hb-ai.heads Spawn some heads near the player
/herobrine cave (player name) hb-ai.cave Create cave near the player. (Location must be under the 60th block of height)
/herobrine curse (player name) hb-ai.curse Curse the player
/herobrine burn (player name) hb-ai.burn Burn the player
/herobrine allworlds hb-ai.allworlds Adds all server worlds to the list of worlds that Herobrine is permitted to enter
- hb-ai.ignore Herobrine will ignore all players that have this permission set
